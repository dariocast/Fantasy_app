# Types

Centralized TypeScript type definitions and interfaces representing the application's domain model.

- **Purpose**: Provides a single source of truth for the data structures used across the frontend and backend, ensuring type safety.
- **What it wraps**: Domain entities for leagues, players, matches, and fantasy team management.
- **Main Entry Point**: `src/types/index.ts`

## Key Files

- `src/types/index.ts` — Contains core interfaces including `User`, `League`, `Player`, `Match`, and `FantasyTeam`, along with complex nested settings for tournament rules and scoring bonuses.
