# Project Context

A mobile application designed for friends to manage their own fantasy football leagues (Fantalega).

## Goals & Success Criteria

- Provide an intuitive interface for managing lineups and predictions.
- Ensure real-time updates for match centers and standings.
- Facilitate league administration for tournament organizers.

## Requirements

### Must Have
- User authentication.
- League creation and joining.
- Lineup submission.
- Match results and standings.

### Should Have
- Player statistics viewer.
- Real-time match notifications.
- Team profile customization.
