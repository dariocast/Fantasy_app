# Styles

Centralized styling constants and shared React Native `StyleSheet` definitions.

- **Purpose**: Ensures visual consistency across the application by providing reusable style patterns for common UI components.
- **What it wraps**: React Native `StyleSheet` API and platform-specific layout logic.
- **Main Entry Point**: `src/styles/commonStyles.ts`

## Key Files

- `src/styles/commonStyles.ts:commonStyles` — Contains shared styles for keyboard avoiding views, modal overlays, responsive grids, and specialized components like the formation field slot.
