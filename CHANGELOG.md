# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-04-24
### Added
- Inizializzazione dell'app "Fantasy" (precedentemente Fantalega).
- Dashboard completa per la gestione dei tornei (gironi, eliminazione diretta, ecc.).
- Match Center per le statistiche dal vivo delle partite.
- Modalità Fantacalcio con gestione rose, formazioni e bonus personalizzati.
- Sincronizzazione in tempo reale con Supabase.
- Funzionalità di eliminazione account sicura per compliance App Store.
- Supporto per la modifica del profilo utente.
- Configurazione EAS (Expo Application Services) per build di produzione.
